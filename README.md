# NBA_Player_Data_Analysis

The primary aim of this data mining task is to reasonably forecast whether the rookie will enjoy
a career that lasts more than 5 years based on their first-year performance statistics which
includes various important attributes. This analysis will be instrumental to better understand
the factors that contribute significantly to a durable NBA career and aims to help the team
owners and managers to make informed decisions regarding their investment into the most
important asset, which is a player. The dataset boasts various attributes to help us classify the
entities.
